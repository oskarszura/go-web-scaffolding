# Golang Web Scaffolding

## About the project
A complete, full-stack, generic website scaffolding project for Go based websites.

**Reason:** Built just for fun and exercise.

## Stacks:

**Front-end stack:**
* SASS
* Compass 
* Twitter's bootstrap (SASS edition)
* ReactJS
* RxJS
* Ramda

**Back-end stack:**
* Golang
